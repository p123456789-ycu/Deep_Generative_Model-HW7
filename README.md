# ⚽ FIFA World Cup 2026 Travel Agent

An AI-powered travel planning application for the FIFA World Cup 2026, combining RAG, Multi-Agent LLM, and ControlNet image generation.

---

## System Architecture

```
User Input (Gradio UI)
        │
        ▼
   RAG Retrieval
   (FAISS + SentenceTransformer)
        │
        ▼
  Orchestrator (Parallel)
  ┌─────┬──────┬───────┐
  │     │      │       │
  ▼     ▼      ▼       │
Itinerary Food  Budget  │
Agent   Agent  Agent    │
  │     │      │        │
  └─────┴──────┘        │
        │               │
        ▼               ▼
   Travel Plan    Landmark Fetcher
        │               │
        └──────┬─────────┘
               ▼
      ControlNet Image Generator
      (SD v1.5 + Canny ControlNet)
               │
               ▼
         Gradio Output
    (Itinerary / Food / Budget / Poster)
```

### Components

| File | Role |
|------|------|
| `generate_kb.py` | Fetch Wikipedia data for 16 host cities, generate knowledge base `.txt` files |
| `build_index.py` | Encode documents and build FAISS vector index |
| `rag.py` | Load FAISS index, retrieve relevant context for LLM |
| `orchestrator.py` | Run 3 agents in parallel using `ThreadPoolExecutor` |
| `agents/itinerary_agent.py` | Day-by-day itinerary planning agent |
| `agents/food_agent.py` | Food & dining recommendation agent |
| `agents/budget_agent.py` | Budget estimation agent |
| `llm_agent.py` | Generate image prompt from itinerary |
| `landmark_fetcher.py` | Fetch city landmark image from web (with local fallback) |
| `image_generator.py` | ControlNet Canny + SD v1.5 poster generation |
| `app.py` | Gradio UI entry point |

## Requirements

- Python 3.10+
- NVIDIA GPU with CUDA (recommended, CPU fallback supported)
- Ollama installed and running locally
- Llama3 model pulled in Ollama

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/worldcup-travel-agent.git
cd worldcup-travel-agent
```

### 2. Create conda environment

```bash
conda create -n travel_agent python=3.10
conda activate travel_agent
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Install and start Ollama

Download Ollama from https://ollama.com and run:

```bash
ollama pull llama3
ollama serve
```

---

## Running the App

Execute in order:

### Step 1 — Build Knowledge Base

```bash
python generate_kb.py
```

Fetches Wikipedia data for all 16 FIFA World Cup 2026 host cities and saves to `kb/cities/`.

### Step 2 — Build FAISS Index

```bash
python build_index.py
```

Encodes documents and saves the vector index to `data/worldcup.index`.

### Step 3 — Launch the App

```bash
python app.py
```

Open your browser at `http://localhost:7860`

## Features

- **RAG Knowledge Base** — FAISS-based retrieval over 16 host cities + tournament schedule
- **Multi-Agent LLM** — 3 specialized agents running in parallel (Itinerary / Food / Budget)
- **ControlNet Image Generation** — City landmark → Canny edge detection → AI travel poster
- **Gradio UI** — Tab-based interactive interface
